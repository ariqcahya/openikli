# Code of Conduct

OpenSource-IKLI adalah project open-source yang ditujukan untuk membantu daerah, instansi, peneliti, dan komunitas.

Kami mengharapkan semua kontributor untuk:

- bersikap sopan;
- menghargai pendapat berbeda;
- memberi kritik secara konstruktif;
- menghindari ujaran kebencian;
- menjaga diskusi tetap relevan;
- tidak menyalahgunakan data publik atau data responden.

Pelanggaran dapat dilaporkan kepada maintainer project.
